Family Travel Tracker

This side project was something I made while learning how to use HTML, CSS, and JavaScript. It uses a PostgreSQL database to keep track of user data and allows the user to switch between family member tabs and add to a list of countries they've visited. The database tracks a different list of countries for each person.
